Instructions

FretboardTunings is a web application which displays the fretboard notes of a guitar. This app allows you to modify the tuning by clicking on an open string to change the notes, making it useful as a visual reference for finding notes and chords while using alternative guitar tunings.

- Left-click on an open string to lower the note by a semitone, or right-click to increase it.

- Select from a list of presets to quickly find a commonly used tuning.
